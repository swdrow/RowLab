<task type="auto">
  <name>Task 3: Create useFeaturePreference hook</name>
  <files>src/v2/hooks/useFeaturePreference.ts</files>
  <action>
Create convenience hooks for feature access:

1. `useFeaturePreference(featureId: FeatureId)` hook:
   - Returns { enabled: boolean, toggle: () => void, isCore: boolean }
   - enabled: result of isFeatureEnabled
   - toggle: calls toggleFeature (no-op if core)
   - isCore: true if featureId is in CORE_FEATURES

2. `useFeature(featureId: FeatureId)` simpler hook:
   - Returns just boolean (enabled state)
   - Shorthand for components that just need to conditionally render

3. `useFeatureGroup(groupId: 'core' | 'advanced')` hook:
   - Returns array of feature configs for that group with current enabled state

4. Export all hooks from index file.

Add JSDoc examples showing usage:
```tsx
// Simple conditional rendering
const showMatrixRacing = useFeature('matrix-seat-racing');
if (!showMatrixRacing) return null;

// With toggle capability
const { enabled, toggle, isCore } = useFeaturePreference('recruiting');
```
