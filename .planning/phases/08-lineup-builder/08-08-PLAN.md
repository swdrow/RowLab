  <name>Task 2: Create marginCalculations utility</name>
  <files>src/v2/utils/marginCalculations.ts</files>
  <action>
Create utility functions for margin calculations:

1. `calculateMarginMeters(timeA: number, timeB: number, winnerSpeed: number)`:
   - Calculate distance gap in meters
   - timeDelta = |timeA - timeB| (seconds)
   - distanceGap = timeDelta * winnerSpeed (m/s)
   - Return { distanceMeters, timeDelta, leader: 'A' | 'B' }

2. `calculateBoatLengths(distanceMeters: number, boatClass: string)`:
   - Shell lengths in meters:
     - '8+': 18.0
     - '4+': 13.2
     - '4-': 13.0
     - '2+': 10.0
     - '2-': 10.0
     - '2x': 10.0
     - '1x': 8.2
   - Return distanceMeters / shellLength

3. `estimateSpeed(time: number, distance: number)`:
   - speed = distance / time (m/s)
   - Common distances: 2000m, 1000m, 500m

4. `formatMargin(boatLengths: number)`:
   - < 0.1: "Dead heat"
   - < 0.25: "Canvas"
   - < 0.5: "1/4 length"
   - < 1.0: "1/2 length"
   - >= 1.0: "X.X lengths"

5. `getShellImage(boatClass: string)`:
   - Map boat class to SVG path
   - Return '/images/shells/eight-plus.svg' etc.

Per MARG-03: "System calculates distance gap from time delta and winner speed"
Per MARG-04: "System displays margin in boat lengths (scaled to shell type)"
  </action>
