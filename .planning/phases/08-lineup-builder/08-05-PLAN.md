<task type="auto">
  <name>Task 1: Create useLineups TanStack Query hooks</name>
  <files>src/v2/hooks/useLineups.ts</files>
  <action>
Create TanStack Query hooks for lineup API operations:

1. `useLineups()` - Fetch all lineups for team
   - Query key: ['lineups']
   - Calls GET /api/v1/lineups via authenticatedFetch

2. `useLineup(lineupId)` - Fetch single lineup with assignments
   - Query key: ['lineups', lineupId]
   - Calls GET /api/v1/lineups/:id
   - Enabled only when lineupId is provided

3. `useSaveLineup()` - Mutation to save new lineup
   - Calls POST /api/v1/lineups
   - Invalidates ['lineups'] on success
   - Body: { name, notes, assignments }

4. `useUpdateLineup()` - Mutation to update existing lineup
   - Calls PATCH /api/v1/lineups/:id
   - Invalidates ['lineups', lineupId] on success

5. `useDuplicateLineup()` - Mutation to duplicate lineup
   - Calls POST /api/v1/lineups/:id/duplicate
   - Body: { name: newName }
   - Invalidates ['lineups'] on success

6. `useDeleteLineup()` - Mutation to delete lineup
   - Calls DELETE /api/v1/lineups/:id
   - Invalidates ['lineups'] on success

Pattern follows existing hooks (useErgTests.ts, useAthletes.ts).
Use authenticatedFetch from authStore for all API calls.
  </action>
