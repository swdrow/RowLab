**Add Whiteboard model** for daily team communications:
```prisma
model Whiteboard {
  id          String   @id @default(uuid())
  teamId      String
  date        DateTime @db.Date
  content     String   @db.Text
  authorId    String
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt

  team        Team     @relation(fields: [teamId], references: [id], onDelete: Cascade)
  author      User     @relation(fields: [authorId], references: [id])

  @@unique([teamId, date])
  @@index([teamId])
  @@index([date])
  @@map("whiteboards")
}
```

**Add DashboardPreferences model** for user dashboard customization:
```prisma
model DashboardPreferences {
  id              String   @id @default(uuid())
  userId          String   @unique
  pinnedModules   Json     @default("[]")   // Array of module IDs
  hiddenSources   Json     @default("[]")   // Array of source names to hide
  headlineType    String?                   // Override for headline widget type
  layout          Json     @default("{}")   // Custom layout configuration
  createdAt       DateTime @default(now())
  updatedAt       DateTime @updatedAt

  user            User     @relation(fields: [userId], references: [id], onDelete: Cascade)

  @@map("dashboard_preferences")
}
```

**Add Activity model** for unified activity feed with source deduplication:
```prisma
model Activity {
  id              String   @id @default(uuid())
  userId          String
  teamId          String?
  source          ActivitySource
  sourceId        String?         // External ID (C2 logbook ID, Strava activity ID)
  type            String          // rowing, cycling, running, strength, etc.
  title           String?
  date            DateTime
  durationSeconds Int?
  distanceMeters  Int?
  calories        Int?
  avgHeartRate    Int?
  metadata        Json     @default("{}")  // Source-specific data
  createdAt       DateTime @default(now())
  updatedAt       DateTime @updatedAt

  user            User     @relation(fields: [userId], references: [id], onDelete: Cascade)
  team            Team?    @relation(fields: [teamId], references: [id], onDelete: Cascade)

  @@unique([source, sourceId])  // Deduplication key
  @@index([userId])
  @@index([teamId])
  @@index([date])
  @@map("activities")
}

enum ActivitySource {
  CONCEPT2
  STRAVA
  MANUAL
  FIT_IMPORT
}
```
