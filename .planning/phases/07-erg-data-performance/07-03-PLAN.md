Create a line chart showing erg test progress over time using recharts:

**Chart Features:**
- X-axis: Date (formatted as MMM DD or MMM YYYY depending on range)
- Y-axis: Time in seconds (formatted as MM:SS)
- Line for each test type (filterable)
- Tooltip showing: Date, Time, Split, Watts
- Optional trend line showing improvement

**Props interface:**
```typescript
interface ErgProgressChartProps {
  tests: ErgTest[];
  testType?: TestType;  // Filter to single type, or show all
  height?: number;  // Default 300
  showTrendLine?: boolean;
  className?: string;
}
```

**Implementation with recharts:**
```tsx
import {
  LineChart, Line, XAxis, YAxis, CartesianGrid,
  Tooltip, ResponsiveContainer, Legend
} from 'recharts';

// Transform tests to chart data format
const chartData = tests
  .filter(t => !testType || t.testType === testType)
  .sort((a, b) => new Date(a.testDate).getTime() - new Date(b.testDate).getTime())
  .map(t => ({
    date: new Date(t.testDate).getTime(),
    time: t.timeSeconds,
    formattedDate: formatDate(t.testDate),
    formattedTime: formatTime(t.timeSeconds),
    testType: t.testType,
  }));
```

**Styling:**
- Use v2 design tokens for colors
- Grid lines subtle (stroke opacity 0.1)
- Tooltip styled to match v2 theme
- Responsive container for auto-sizing

**Y-axis inversion consideration:** Lower time = better, so chart should show improvement as upward trend. Either:
- Invert Y-axis (lower values at top) OR
- Use a derived "performance" score instead of raw time

Recommend: Keep time on Y with lower at bottom (standard), but add visual cue (green for improvement) in tooltip.

**Empty state:** Show "No tests to display" message if tests array empty.
