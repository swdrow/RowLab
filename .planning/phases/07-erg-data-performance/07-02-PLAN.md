Props:
```typescript

  onSubmit: (data: CreateErgTestInput | UpdateErgTestInput) => void;
  onCancel: () => void;
  athletes: Athlete[];
  isSubmitting?: boolean;
}
```

Add helper for time input: Allow "6:30.5" format, convert to 390.5 seconds.
Calculate splitSeconds automatically: timeSeconds / (distanceM / 500) if distanceM provided.

**Auto-calculation between Split/Watts** (UX enhancement):
- Formula: `watts = 2.80 / (pace^3)` where pace is split in seconds per 500m
- Inverse: `pace = (2.80 / watts)^(1/3)`
- When user enters watts, auto-populate split field (and vice versa)
- Use `useEffect` to recalculate when either field changes
- Allow manual override (don't lock fields)
