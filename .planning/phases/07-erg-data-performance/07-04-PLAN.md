  <action>
Create CSV parsing utilities for erg tests, following csvParser.ts pattern:

**1. parseErgCSV(file: File): Promise<ParseResult>**
- Uses PapaParse with same config (header: true, dynamicTyping: true)
- Worker threads for files > 500KB
- Returns { data: Record<string, unknown>[], errors: ParseError[], meta }

**2. autoMapErgColumns(headers: string[]): ColumnMapping**
- Maps CSV headers to erg test fields using fuzzy matching
- Required mappings: athlete (name or ID), testType, testDate, time
- Optional mappings: distance, split, watts, strokeRate, weight, notes

**Column aliases to handle:**
```typescript
const COLUMN_ALIASES = {
  athleteId: ['athlete_id', 'athlete', 'rower', 'name', 'athlete_name', 'rower_name'],
  athleteName: ['athlete_name', 'name', 'full_name', 'rower', 'rower_name'],
  testType: ['test_type', 'type', 'test', 'distance', 'piece', 'event'],
  testDate: ['test_date', 'date', 'test_day', 'when'],
  timeSeconds: ['time', 'time_seconds', 'total_time', 'finish_time', 'result'],
  distanceM: ['distance', 'distance_m', 'meters', 'dist'],
  splitSeconds: ['split', 'split_seconds', 'pace', '500_split', 'avg_split'],
  watts: ['watts', 'power', 'avg_watts', 'average_watts'],
  strokeRate: ['stroke_rate', 'spm', 'rate', 'strokes_per_minute', 'sr'],
  weightKg: ['weight', 'weight_kg', 'body_weight', 'bw'],
  notes: ['notes', 'comments', 'remarks'],
};
```

**3. parseTimeToSeconds(timeStr: string): number | null**
- Handles multiple formats:
  - "6:30.5" -> 390.5
  - "6:30" -> 390
  - "390.5" -> 390.5 (already seconds)
  - "1:06:30" -> 3990 (hours:minutes:seconds)

**4. parseTestType(value: string): TestType | null**
- Normalize: "2k", "2K", "2000m", "2000" -> "2k"
- Similar for 6k, 30min, 500m

**5. validateErgTestRow(row: Record<string, unknown>, mapping: ColumnMapping, athletes: Athlete[]): ValidationResult**
- Zod schema for validation
- Athlete lookup by name or ID (fuzzy match on name)
- Date validation
- Time validation
- Returns { valid: boolean, data?: CreateErgTestInput, errors?: string[] }

**6. validateAllErgRows(rows, mapping, athletes): { valid: ValidRow[], invalid: InvalidRow[] }**
