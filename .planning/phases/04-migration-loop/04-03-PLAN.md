<task type="auto">

  <action>
Create server/routes/v1/oarSets.js following shells.js pattern exactly:

**GET /api/v1/oar-sets**
- Middleware: authenticateToken, teamIsolation
- Optional query filter: ?type=SWEEP or ?type=SCULL
- Returns { success: true, data: { oarSets: [...] } }

**GET /api/v1/oar-sets/:id**
- Middleware: authenticateToken, teamIsolation
- Validation: param('id').isUUID()
- Returns oar set or 404 NOT_FOUND

**POST /api/v1/oar-sets**
- Middleware: authenticateToken, teamIsolation, requireRole('OWNER', 'COACH')
- Validation:
  - body('name').trim().isLength({ min: 1, max: 100 })
  - body('type').isIn(['SWEEP', 'SCULL'])
  - body('count').isInt({ min: 1, max: 100 })
  - body('status').optional().isIn(['AVAILABLE', 'IN_USE', 'MAINTENANCE', 'RETIRED'])
  - body('notes').optional().trim().isLength({ max: 500 })
- Returns 201 with oar set data

**PATCH /api/v1/oar-sets/:id**
- Middleware: authenticateToken, teamIsolation, requireRole('OWNER', 'COACH')
- Validation: All fields optional except id
- Returns updated oar set or 404

**DELETE /api/v1/oar-sets/:id**
- Middleware: authenticateToken, teamIsolation, requireRole('OWNER', 'COACH')
- Returns { success: true, data: { message: 'Oar set deleted' } } or 404

Mount in server/index.js alongside shells:
```javascript
import oarSetRoutes from './routes/v1/oarSets.js';
app.use('/api/v1/oar-sets', apiLimiter, oarSetRoutes);
```
