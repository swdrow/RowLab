<task type="auto">
  <name>Task 2: Create whiteboard routes and mount</name>
  <files>server/routes/whiteboards.js, server/index.js</files>
  <action>
Create server/routes/whiteboards.js with endpoints:

**GET /api/v1/whiteboards/latest**
- Middleware: authenticateToken, teamIsolation
- Returns latest whiteboard for team (or 404 with NO_WHITEBOARD code if none)
- Include author { id, name } in response

**GET /api/v1/whiteboards/:id**
- Middleware: authenticateToken, teamIsolation
- Validation: param('id').isUUID()
- Returns specific whiteboard or 404

**POST /api/v1/whiteboards**
- Middleware: authenticateToken, teamIsolation, requireRole('COACH', 'OWNER')
- Validation:
  - body('date').isISO8601().toDate() - YYYY-MM-DD format
  - body('content').isString().trim().notEmpty().isLength({ max: 50000 })
- Uses upsert pattern (creates if not exists, updates if exists for that date)
- Returns 201 with whiteboard data

**DELETE /api/v1/whiteboards/:id**
- Middleware: authenticateToken, teamIsolation, requireRole('COACH', 'OWNER')
- Validation: param('id').isUUID()
- Returns 200 with success message or 404

Follow existing patterns:
- Import validateRequest helper (same pattern as dashboardPreferences.js)
- Use logger.error for error logging
- Standard response format: { success: true, data: { whiteboard } }

Mount in server/index.js:
```javascript
import whiteboardRoutes from './routes/whiteboards.js';
// Mount with apiLimiter
app.use('/api/v1/whiteboards', apiLimiter, whiteboardRoutes);
```
