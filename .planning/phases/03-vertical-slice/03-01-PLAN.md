<task type="auto">
  <name>Task 2: Create QueryClient singleton with default options</name>
  <files>src/v2/queryClient.ts</files>
  <action>
Create `src/v2/queryClient.ts` with QueryClient singleton:

```typescript
import { QueryClient } from '@tanstack/react-query';

// Create client outside component to avoid recreation on render
export const queryClient = new QueryClient({
  defaultOptions: {
    queries: {
      staleTime: 5 * 60 * 1000, // 5 minutes
      gcTime: 10 * 60 * 1000, // 10 minutes (v5 renamed cacheTime)
      retry: 1, // Retry failed requests once
      refetchOnWindowFocus: false, // Don't refetch on tab focus
    },
    mutations: {
      retry: 0, // Don't retry mutations
    },
  },
});
```

Settings rationale (from research):
- staleTime 5min: External APIs (C2, Strava) don't change frequently
- gcTime 10min: Keep data cached for background tabs
- refetchOnWindowFocus false: Prevents API hammering when switching tabs
- retry 1: One retry for network issues, not more (external API rate limits)
  </action>
  <verify>TypeScript compiles without errors: `npx tsc --noEmit src/v2/queryClient.ts`</verify>
  <done>QueryClient singleton exists with appropriate default options for external API data fetching.</done>
</task>
