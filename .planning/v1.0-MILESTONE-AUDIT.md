## Tech Debt Summary

### Phase 2: Foundation

**Light and Field Theme CSS Rendering**
- Dark theme works perfectly
- Light and Field themes set `data-theme` correctly but don't visually apply
- Root cause: CSS variable cascade issue between tokens.css and V2Layout styles
- **Impact:** Low — dark mode is default and works; most users prefer dark mode
- **Resolution:** Post-v1.0 CSS debugging required

### Phase 5: The Flip

**V1-Only Features Not Migrated**
- Lineup builder (complex boat arrangement UI)
- 3D boat view (Three.js visualization)
- Erg data page (detailed erg analytics)
- Seat racing (comparison tools)
- Training plans (builder and assignment)
- Racing page (race day management)
- Communication page (team messaging)
- Advanced page (advanced features)
- Coxswain view (coxswain-specific UI)
- Athletes roster (roster management)
- Settings page (user/team settings)

**Impact:** Medium — users must access these at `/legacy/*` routes
**Mitigation:** V2 VersionToggle allows instant switch to V1

**Legacy Features Discoverability**
- "Legacy Features" section not yet added to V2 sidebar
- Users must know to use toggle or type `/legacy/*` URLs
- **Impact:** Low — toggle is prominent in header
- **Resolution:** Add sidebar section linking to legacy routes

### Total Tech Debt

- **Phase 2:** 2 items (CSS theming)
- **Phase 5:** 2 items (migration scope + discoverability)
- **Total:** 4 items across 2 phases
